# 09 – Checklists, grilles de restitution, lexique rapide

## 1) Checklist technique (avant toute interprétation)
- [ ] Date OK (JJ/MM/AAAA)
- [ ] Heure OK (HH:MM) + source (acte / famille / estimation)
- [ ] Lieu OK (ville + pays) + coordonnées correctes
- [ ] Fuseau / DST OK (heure locale → UT)
- [ ] Réglages : tropical, géocentrique
- [ ] Maisons : Placidus (et contrôle Whole Sign)
- [ ] Angles affichés : ASC/MC
- [ ] Nœud : vrai (ou moyen, choisi)
- [ ] Orbes : majeurs d’abord

## 4) Grille lecture natal (ordre)
1) Angles (ASC/MC/IC/DSC)
2) Soleil + Lune
3) Maître d’ASC
4) Mercure / Vénus / Mars
5) Jupiter / Saturne
6) Transpersonnelles (si pertinentes)
7) Configurations + répétitions
8) Synthèse

## 5) Lexique rapide
- Angularité : présence forte (I/IV/VII/X).
- Dignité : “facilité” structurelle d’une planète.
- Appliquant/séparant : aspect en formation / en éloignement.
- Stellium : concentration (3+ planètes).
- Rétrograde : fonction plus introspective/révisée.
- Whole Sign : 1 maison = 1 signe (à partir du signe ASC).
- Placidus : maisons par semi-arcs (dépend fortement de la latitude).
