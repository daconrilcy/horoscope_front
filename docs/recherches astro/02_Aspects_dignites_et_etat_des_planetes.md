# 02 – Aspects, orbes, dignités et “état” des planètes

## 1) Aspects : angles et sens (le “comment ça interagit”)
Aspects majeurs (à maîtriser en premier) :
- Conjonction (0°) : fusion, concentration, amplification.
- Sextile (60°) : opportunité, facilité active (demande de l’initiative).
- Carré (90°) : tension dynamique, friction, défi.
- Trigone (120°) : fluidité, talent, aisance (peut “dormir”).
- Opposition (180°) : polarité, projection, besoin d’équilibre.

Aspects mineurs (optionnels au début) :
- Quinconce (150°) : ajustement, incompatibilité de registre.
- Semi-carré (45°), sesqui-carré (135°) : micro-tensions.
- Semi-sextile (30°) : continuité faible.

## 2) Orbes recommandés (conservateurs)
Principe : plus l’orbe est serré, plus l’aspect est “fort”. Les luminaires tolèrent plus.

Repères usuels (tu peux ajuster après expérience) :
- Soleil/Lune : 8–10° max (serré = plus fiable)
- Planètes personnelles : 6–8°
- Planètes sociales : 5–6°
- Transpersonnelles : 3–5°
- Aspects mineurs : 1–2° (sinon bruit)

Règle pratique pro : si ton interprétation repose sur des aspects “larges” uniquement, prudence.

## 3) Appliquant / séparant (temporalité interne)
- Appliquant : l’aspect se forme (tension/énergie “qui monte”).
- Séparant : l’aspect s’éloigne (expérience déjà intégrée).
Ce critère est surtout utile en transit/prévisionnel.

## 4) Dignités essentielles (facilité structurelle)
Domicile : planète “chez elle” → fonction plus directe.
Exil : planète “en face” → fonction moins naturelle, apprentissage.
Exaltation : planète “honorée” → puissance, idéalisation.
Chute : opposé à exaltation → expression moins stable.

Table simplifiée (à mémoriser, version classique) :
- Soleil : domicile Lion, exalt Bélier, exil Verseau, chute Balance
- Lune : domicile Cancer, exalt Taureau, exil Capricorne, chute Scorpion
- Mercure : domicile Gémeaux/Vierge, exalt Vierge, exil Sagittaire/Poissons, chute Poissons
- Vénus : domicile Taureau/Balance, exalt Poissons, exil Scorpion/Bélier, chute Vierge
- Mars : domicile Bélier/Scorpion, exalt Capricorne, exil Balance/Taureau, chute Cancer
- Jupiter : domicile Sagittaire/Poissons, exalt Cancer, exil Gémeaux/Vierge, chute Capricorne
- Saturne : domicile Capricorne/Verseau, exalt Balance, exil Cancer/Lion, chute Bélier

## 5) Dignités accidentelles (visibilité / efficacité)
- Angularité : maisons I, IV, VII, X → très “opérationnel”.
- Vitesse : planète rapide = plus active ; lente = plus intériorisée/collective.
- Rétrogradation : énergie plus introspective, révision, parfois décalage.
- Sous les rayons / combustion / cazimi (Soleil) :
  - “Sous les rayons” : trop proche du Soleil → visibilité réduite.
  - Combustion : très proche → fonction “brûlée”/difficile à exprimer.
  - Cazimi : extrêmement proche (dans le cœur du Soleil) → intensification rare.

(Pratique : la plupart des logiciels indiquent ces états. L’important est de comprendre le sens : visibilité et accès à la fonction.)

## 6) Configurations à reconnaître (patterns)
- Stellium : 3+ planètes regroupées (signe/maison) → sur-investissement.
- T-carré : deux oppositions + carré → tension structurante, moteur.
- Grand trigone : fluidité, talent ; risque d’inertie.
- Cerf-volant (kite) : grand trigone “activé” par une opposition.
- Yod (doigt de Dieu) : deux quinconces convergeant → ajustements.

## 7) Hiérarchie utile (pour éviter de “sur-interpréter”)
1) Angles + luminaires + maître d’ASC
2) Conjonctions / oppositions / carrés serrés
3) Trigones / sextiles serrés
4) Configurations (T-carré, stellium…)
5) Aspects mineurs (si très serrés et pertinents)

## 8) Traduction pro (formulation)
Un aspect n’est pas “bon” ou “mauvais” : c’est une dynamique.
- Carré : “un sujet de croissance par friction”
- Opposition : “un équilibre à construire”
- Trigone : “un talent naturel à activer consciemment”
