# 01 – Langage astro : signes, planètes, maisons, angles, dominantes

## 1) Les 12 signes : élément + modalité + maître (table de base)
Chaque signe = un style d’expression (signe) appliqué à une fonction (planète) dans un domaine (maison).

| Signe | Élément | Modalité | Maître (trad.) | Mots-clés (sobres) |
|---|---|---|---|---|
| Bélier | Feu | Cardinal | Mars | initier, oser, trancher |
| Taureau | Terre | Fixe | Vénus | stabiliser, posséder, sentir |
| Gémeaux | Air | Mutable | Mercure | relier, apprendre, varier |
| Cancer | Eau | Cardinal | Lune | protéger, contenir, appartenir |
| Lion | Feu | Fixe | Soleil | rayonner, créer, s’affirmer |
| Vierge | Terre | Mutable | Mercure | trier, servir, améliorer |
| Balance | Air | Cardinal | Vénus | harmoniser, négocier, choisir |
| Scorpion | Eau | Fixe | Mars (trad.) | intensifier, transformer, lier/délier |
| Sagittaire | Feu | Mutable | Jupiter | élargir, croire, explorer |
| Capricorne | Terre | Cardinal | Saturne | structurer, durer, assumer |
| Verseau | Air | Fixe | Saturne (trad.) | différencier, réformer, penser collectif |
| Poissons | Eau | Mutable | Jupiter (trad.) | dissoudre, imaginer, compatir |

Note : maîtres modernes souvent ajoutés (Scorpion/Pluton, Verseau/Uranus, Poissons/Neptune). Apprends d’abord la logique “traditionnelle”, puis ajoute les modernes.

## 2) Les planètes : fonctions (le “quoi”)
Planètes personnelles (identité, besoins, désir) :
- Soleil : centre, volonté, direction, identité consciente.
- Lune : besoins, sécurité, émotions, mémoire, rythmes.
- Mercure : pensée, langage, apprentissage, échanges.
- Vénus : valeurs, attractivité, lien, plaisir, goût.
- Mars : énergie, désir, action, affirmation, conflit.

Planètes sociales (cadre + expansion) :
- Jupiter : croissance, sens, opportunités, confiance, excès.
- Saturne : limites, responsabilité, structure, temps, maîtrise.

Transpersonnelles (collectif / transformation) :
- Uranus : rupture, individuation, changement, innovation.
- Neptune : idéal, dissolution, inspiration, flou, foi.
- Pluton : intensité, pouvoir, crise, régénération.

## 3) Angles : points structurants (priorité absolue)
- Ascendant (ASC) : interface au monde, style d’entrée en relation, “comment tu démarres”.
- Descendant (DSC) : style de relation, partenaires/miroirs.
- Milieu du Ciel (MC) : direction, visibilité, vocation/social.
- Fond du Ciel (IC) : racines, intimité, base psychique.

Règle pratique : un thème se lit d’abord par les luminaires + ASC/MC.

## 4) Les 12 maisons : domaines (le “où”)
| Maison | Domaine principal | Mots-clés |
|---|---|---|
| I | identité, corps, élan | présence, tempérament |
| II | ressources, valeurs | argent, sécurité matérielle |
| III | apprentissage, entourage | communication, fratrie, mobilité |
| IV | racines, foyer | famille, intériorité |
| V | création, plaisir | amour, enfants, expression |
| VI | travail quotidien | santé hygiène, routines, service |
| VII | relation, contrats | couple, associés, clients |
| VIII | partages, crises | dettes, héritage, intimité, transformation |
| IX | sens, horizons | voyages, études sup., croyances |
| X | carrière, statut | vocation, autorité, réputation |
| XI | réseaux, projets | amis, collectifs, futur |
| XII | retrait, inconscient | solitude, coulisses, dissolution |

## 5) Dominantes (pour prioriser au lieu de “tout lire”)
Dominantes par éléments :
- Feu : agir, entreprendre, se risquer.
- Terre : concrétiser, stabiliser, produire.
- Air : penser, relier, socialiser.
- Eau : ressentir, s’attacher, imaginer.

Dominantes par modalités :
- Cardinal : initier, démarrer, impulser.
- Fixe : tenir, approfondir, stabiliser.
- Mutable : adapter, apprendre, pivoter.

Dominantes par angularité :
- Planètes en maisons I, IV, VII, X (angulaires) = très visibles, très structurantes.
- Succédentes (II, V, VIII, XI) = consolidation.
- Cadentes (III, VI, IX, XII) = diffusion/apprentissage.

## 6) Le maître d’Ascendant (pièce maîtresse)
Étape clé : identifier le maître du signe Ascendant, regarder :
- son signe (comment il fonctionne),
- sa maison (où ça se joue),
- ses aspects (avec quoi il doit composer),
- sa dignité/état (facilité/tension).

## 7) Mémo “phrase de placement”
Pour exprimer un placement :  
“(Planète) = (fonction) s’exprime en (signe) = (style) dans (maison) = (domaine), modulée par (aspects) = (dynamiques).”
