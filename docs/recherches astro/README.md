# Formation complète – Astrologie occidentale tropicale (niveau praticien)

Objectif : être capable de produire un thème natal correct (données + réglages), de l’interpréter de façon structurée, et de réaliser du prévisionnel de base (transits + révolution solaire), avec un cadre déontologique solide.

Cette formation est volontairement **standardisée** sur un référentiel pour éviter de se disperser :
- Zodiaque : **tropical**
- Référentiel : **géocentrique** (positions apparentes)
- Orbes : approche **conservatrice**
- Maisons : **Placidus** (par défaut), avec équivalents **Whole Sign** et **Equal** (pour contrôle / robustesse)
- Nœuds : **vrai** (par défaut) – tu apprendras aussi “moyen”
- Lilith : optionnelle (tu apprendras à ne pas en dépendre)

Contenu des fichiers :
- `00_Orientation_et_reglages.md` : choix d’école, réglages, outils, règles de contrôle
- `01_Langage_astro_signes_planetes_maisons.md` : signes, planètes, maisons, angles, dominantes
- `02_Aspects_dignites_et_etat_des_planetes.md` : aspects, orbes, dignités, états (rétrograde, cazimi, etc.)
- `03_Methode_de_lecture_natal.md` : protocole de lecture + synthèse (structure pro)
- `04_Transits_pratique.md` : transits (technique + temporalité + hiérarchisation)
- `05_Revolution_solaire_pratique.md` : thème annuel (RS) + articulation avec transits
- `06_Progressions_secondaires_option.md` : progressions (option très utile)
- `07_Synastrie_option.md` : relationnel (option)
- `08_Calculs_donnees_ephemerides.md` : données, conversions, calculs, contrôles
- `09_Checklists_et_grilles_de_restitution.md` : checklists + grilles de lecture + lexique rapide

Suggestion d’usage :
1. Lis `00` → règle ton logiciel une fois pour toutes (et sais contrôler).
2. Apprends le langage `01` + `02` (sans interpréter “au hasard”).
3. Applique la méthode `03` sur 10 thèmes tests.
4. Ajoute `04` et `05` (prévisionnel).
5. Option : `06` et `07`.
6. Consolide avec `09` (checklists).

Important : l’astrologie n’est pas une science prédictive au sens expérimental. Dans la pratique, on travaille comme avec un **langage symbolique** : on formule des hypothèses, on teste par le dialogue, on nuance, on garde un cadre éthique strict (notamment santé/juridique/finance).
