# Step 03 - Traceability Matrix

Build the matrix before prose.

```md
| Story | AC / Expected outcome | Initial need source | Implemented evidence | Validation evidence | Status |
|---|---|---|---|---|---|
```

Rules:

- Prefer one row per AC or expected outcome; use a story-level row only when
  outcomes cannot be split meaningfully.
- Evidence cells must cite concrete anchors and what they prove.
- Use only workflow statuses.
- After the matrix, derive evidence gaps and residual risks from weak/missing
  cells; do not bury them elsewhere.
