# Limites avant integration frontend

- Aucun fichier frontend, client genere ou style n'a ete modifie par CS-302.
- Les preuves couvrent le endpoint HTTP public existant avec `TestClient`, les builders publics reels, les plans B2C supportes, les erreurs publiques, OpenAPI et les routes runtime.
- Les exemples JSON sont des artefacts de test backend et ne remplacent pas une validation UX ou une generation de client TypeScript.
- La persistance reste prouvee par le chemin API existant `test_projection_persistence_endpoint.py`; aucune migration ni schema DB n'est change.
